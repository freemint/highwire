Keyboard Shortcuts / Hot keys

 
General:
CTRL+I  About Highwire alert
CTRL+Q  Quit Highwire
Help / F1  Open Help
 
Window:
CTRL+N  Open a new window
CTRL+U  Close window
CTRL+W  Cycle window
TAB  Cycle the active frames (inside the top window). That's the frame that will receive keyboard input, like scrolling.
Arrow-Keys  Scrolls the active frame by one line/row into the appropriated direction (if not already at border)
 
Navigation:
CTRL+O  Open file selector
SHIFT+CTRL+O  Open URL box
Undo  Go back in file history
SHIFT+Undo  Go forward in file history
 
Display:
CTRL+ R / F5  Reload
CTRL+ +  Increase the fontsize for displaying
CTRL+ -  Decrease the fontsize for displaying
1  Display current page with character set ISO 8859-1 / Latin-1
2  Display current page with character set ISO 8859-2 / Latin-2
F  Display current page with character set ISO 8859-15 / Latin-8
U  Display current page with encoding UTF-8, Unicode
A  Display current page with character set Atari ST system font
M  Display current page with character set Apple Macintosh Roman
 
Assorted:
F7 Toggle logging to console
F8 Toggle pictures or alternative text
Arrow keys Scroll Frame
 
Forms:
Backspace Remove character to left of cursor
Delete Remove character to right of cursor
Shift Backspace Remove all characters to left of cursor
Shift Delete Remove all characters to right of cursor
CTRL+ Backspace Remove all characters to left of cursor
CTRL+ Delete Remove all characters to right of cursor
Escape Remove all text in text area
