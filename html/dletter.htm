A Message from the HighWire Development Team

Welcome to the latest release of HighWire, version 0.2.6

It has been several months since the last public release and for that we are sorry. Between real world concerns eating up our time and some very tricky to find bugs, this version has taken us much longer to put together than any of us would have wished. However the long process was not without results. Several low level problems have been eliminated and a number of new features have been implemented. We hope that you will find this version to be an enjoyable improvement over the last version, and we promise that the next release will not take as long to get together as this version.

Main changes and additions in this versions:

  • Download manager implemented for 'Save Target As' popup. Can handle up to four simultaneous connections
  • Support added for HTTP Authentication (RFC2069 Basic only)
  • The URL input field got an own history for the ten last URLs typed in by the user.
  • Improvements in the removal of control characters in Copy&Paste of URLs
  • Table attributes COLSPAN=0 and ROWSPAN=0 implemented
  • A few Memory bugs were squashed
  • Loading strategy improved for fewer timeouts
  • Javascript and HTML comment filtering improved
  • Code adapted for gemlib version >= 0.43 and inline asm adapted for gcc 3.
  • Several more (but less obvious) bugs fixed.
As usual, more details of changes can be seen in the Change.Log file.

– And as always for your help and support, Thank you,

– HighWire Development Team
