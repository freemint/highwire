A Message from the HighWire Development Team

Welcome to the latest release of HighWire, version 0.2.7

What would Christmas be without a New HighWire release under your virtual Fuji tree.

It has been a tradition to do a release at Chistmas and we didn't want to let anyone be disappointed. Development over the last month has proceded at a vigourous pace, with alot of new supports being added to HighWire. The CSS support is more robust and several small bugs have been eliminated.

However all this development, like any DIY project has left some rough edges. You can expect some pages to look better than they have ever looked before, but you can also expect some pages to show problems that weren't there before. We discovered some errors in the CSS parsing routines that are causing some styles not to be applied. This can cause things like links to be invisible on their background colors or other similar effects. We have identified what the problem is, but unfortunately we were unable to fix it before this release.

All warnings out of the way. We think you will find this Christmas snapshot version to be a nice update to the browser. Most notably the often requested ability to save images off of websites has been implemented now via the right pop up menu. The additions to CSS styles to more tags has quite dramatically changed to look and feel of many sites. We hope you will enjoy our small holiday gift and that you find the improvements to outweigh the problems.

Main changes and additions in this versions:

  • 'Save As' implemented for Images
  • Css styles applied to many new tags
  • Numerous improvements on the CSS style supports
  • Percentage Font Size bug fixed
  • Table Cell 'nowrap' bug fixed
  • Several more (but less obvious) bugs fixed.
As usual, more details of changes can be seen in the Change.Log file.

– And as always for your help and support, Thank you,

– HighWire Development Team
